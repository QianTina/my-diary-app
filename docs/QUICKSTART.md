# 快速开始指南

## 🚀 5 分钟上手

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

打开浏览器访问 `http://localhost:5173`

### 3. 开始使用
1. 在"写新日记"区域输入内容
2. 添加标签（可选）
3. 点击"立即记录"保存

就这么简单！🎉

---

## 📝 基础操作

### 写日记
```
今天天气**很好**，心情*不错*！

## 做了什么
- 早上跑步
- 下午看书
```

点击"👁️ 预览"查看 Markdown 效果。

### 添加标签
在标签输入框输入 `开心`，按 `Enter` 添加。

### 搜索日记
在搜索框输入关键词，实时过滤结果。

### 筛选标签
点击标签云中的标签，筛选相关日记。

---

## 💾 数据备份

### 导出数据
1. 点击右上角"⚙️ 设置"
2. 选择"📥 导出为 JSON"
3. 文件自动下载

### 导入数据
1. 点击"📤 从 JSON 导入"
2. 选择之前导出的 JSON 文件
3. 确认导入

---

## 🌐 云端同步（可选）

### 配置 Supabase
1. 创建 Supabase 项目：https://supabase.com
2. 创建 `.env` 文件：
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
```
3. 在 Supabase SQL Editor 执行 `sql/schema.sql`
4. 重启应用

数据将自动同步到云端！☁️

## 🎯 进阶功能

### Markdown 语法
- 标题：`# H1`, `## H2`
- 粗体：`**粗体**`
- 斜体：`*斜体*`
- 列表：`- 项目`
- 链接：`[文字](URL)`
- 代码：`` `代码` ``
- 引用：`> 引用`

### 标签管理
- 添加：输入标签名，按 `Enter`
- 删除：点击标签上的 `×`
- 筛选：点击标签云中的标签
- 多选：可以选择多个标签

### 草稿保存
输入的内容会自动保存，刷新页面后自动恢复。

## ❓ 常见问题

**Q: 数据存储在哪里？**  
A: 默认存储在浏览器 LocalStorage，可选配置 Supabase 云端同步。

**Q: 数据会丢失吗？**  
A: LocalStorage 数据不会丢失（除非清除浏览器数据），建议定期导出备份。

**Q: 支持移动端吗？**  
A: 支持响应式设计，可在移动端浏览器使用。

**Q: 可以导出为 PDF 吗？**  
A: 目前支持 JSON 和 Markdown 格式，Markdown 可以用其他工具转换为 PDF。
